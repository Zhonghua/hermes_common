Python documentation¶

This section documents the Python interface of hermes_common. If you want to use hermes_common from C++, see the C++ documentation.

class _hermes_common.Matrix¶

Bases: object

add()¶

Adds the value v to self[m, n].

Example:

>>> from _hermes_common import CooMatrix
>>> a = CooMatrix(5)
>>> a.add(1, 3, 4.5)
get_size()¶

Returns the dimension of the square matrix.

Example:

>>> from _hermes_common import CooMatrix
>>> a = CooMatrix(5)
>>> a.get_size()
5

row_col_data¶

Returns (row, col, data) arrays.

Example:

>>> from _hermes_common import CooMatrix
>>> a = CooMatrix(5)
>>> a.add(1, 3, 4.5)
>>> a.add(2, 3, 3.5)
>>> a.add(0, 2, 1.5)
>>> a.row_col_data
(array([1, 2, 0], dtype=int32), array([3, 3, 2], dtype=int32), array([ 4.5,  3.5,  1.5]))
to_scipy_coo()¶

Converts itself to the scipy sparse COO format.

Example:

>>> from _hermes_common import CooMatrix
>>> a = CooMatrix(5)
>>> a.add(1, 3, 4.5)
>>> a.add(2, 3, 3.5)
>>> a.add(0, 2, 1.5)
>>> a
<_hermes_common.CooMatrix object at 0x7f50d30d3108>
>>> a.to_scipy_coo()
<5x5 sparse matrix of type '<type 'numpy.float64'>'
    with 3 stored elements in COOrdinate format>
>>> type(a.to_scipy_coo())
<class 'scipy.sparse.coo.coo_matrix'>
class _hermes_common.CSRMatrix¶

Bases: _hermes_common.SparseMatrix

Represents a CSR Matrix.

You have to create some other matrix first (in most cases Coo) and then convert it.

Example:

>>> from _hermes_common import CooMatrix, CSRMatrix
>>> c = CooMatrix(5)
>>> c.add(2, 3, 1.5)
>>> c.add(1, 2, 2.5)
>>> m = CSRMatrix(c)
